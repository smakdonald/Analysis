Theorem 1.21. Properties of Ordered Fields.

The following statements are true in every ordered field.
  1. If \(x > 0\) then \(- x < 0\text{,}\) and vice versa.
  2. If \(x > 0\) and \(y < z\) then \(xy < xz\text{.}\)
  3. If \(x < 0\) and \(y < z\) then \(xy > xz\text{.}\)
  4. If \(x > 0\) then \(x^2 > 0\text{.}\) In particular, \(1 > 0\text{.}\)
  5. If \(O < x < y\) then \(O < \frac 1/y < \frac 1/x\text{.}\)
