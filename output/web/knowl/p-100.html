A Dedekind cut in \(\mathbb{Q}\) is a set \(x\subseteq \mathbb{Q}\) (we're thinking of \(x\) as a real number) such that
  1. \(\displaystyle x\neq\emptyset, x\neq \mathbb{Q}\)
  2. If \(p\in\mathbb{Q},q\in x\) with \(p<q\text{,}\) then \(p\in x\)
  3. \(\nexists\) maximal element in \(x\text{,}\) ie if \(p\in x\,\exists\, q\in x\) with \(p<q\text{.}\)
