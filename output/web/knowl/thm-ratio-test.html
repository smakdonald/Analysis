Theorem 3.11.

Suppose \((a_n)_{n=1}^{\infty}\subseteq \mathbb{C}\setminus\{0\}\text{.}\)
  1. If \(\limsup_{n\rightarrow\infty}\frac{|a_{n+1}|}{|a_n|}<1\) then \(\sum_{n=1}^{\infty}a_n\) converges absolutely.
  2. If \(\liminf_{n\rightarrow\infty}\frac{|a_{n+1}|}{|a_n|}>1\) then \(\sum_{n=1}^{\infty}a_n\) diverges.

Proof.
