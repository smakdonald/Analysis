  1. \(\mathcal{B}(X)\) is a vector space
  2. For \(f,g\in\mathcal{B}(X),\alpha\in\mathbb{C}\text{,}\)
    1. \(\displaystyle \left|{\left|{{f}}\right|}\right|_\infty=0\iff f=0\)
    2. \(\displaystyle \left|{\left|{{\alpha f}}\right|}\right|_\infty=\left|{\alpha}\right|\left|{\left|{{f}}\right|}\right|_\infty\)
    3. \(\left|{\left|{{f+g}}\right|}\right|_\infty\leq \left|{\left|{{f}}\right|}\right|_\infty+\left|{\left|{{g}}\right|}\right|_\infty\) (triangle inequality)
    4. \(\displaystyle \left|{\left|{{\overline{f}}}\right|}\right|_\infty=\left|{\left|{{f}}\right|}\right|_\infty\)
    5. \(\left|{\left|{{fg}}\right|}\right|_\infty\leq \left|{\left|{{f}}\right|}\right|_\infty\left|{\left|{{g}}\right|}\right|_\infty\) (since \(\left|{fg(x)}\right|=\left|{f(x)}\right|\left|{g(x)}\right|\leq \left|{\left|{{f}}\right|}\right|_\infty\left|{\left|{{g}}\right|}\right|_\infty \forall x\in X\text{.}\))
  3. The function \(d_\infty:\mathcal{B}(x)\times \mathcal{B}(x)\rightarrow[0,\infty)\) by \(d_\infty(f,g)=\left|{\left|{{f-g}}\right|}\right|_\infty\) is a metric on \(\mathcal{B}(X)\text{.}\)
  4. if \((f_n)_{n=1}^\infty\subseteq\mathcal{B}(X)\) and \(f\in\mathcal{B}(X)\text{,}\) then \(f_n\rightarrow f\) in the metric \(d_\infty\iff f_n\rightarrow f\) uniformly on \(X\text{.}\) (both conditions mean
    \begin{equation*} \lim_{n\rightarrow\infty}d_\infty(f_n,f)=\lim_{n\rightarrow\infty}\sup_{x\in X}\left|{f_n(x)-f(x)}\right|=0) \end{equation*}
  5. Addition, multiplication, and conjugation are continuous on \(\mathcal{B}(X)\text{:}\) ie if \((f_n)_{n=1}^\infty,(g_n)_{n=1}^\infty\subseteq\mathcal{B}(X)\text{,}\) \(f_n\rightarrow f\) and \(g_n\rightarrow g\) uniformly then
    \begin{equation*} \begin{aligned} f_n+g_n&\rightarrow f+g\\ f_ng_n&\rightarrow fg\\ \overline{f_n}&\rightarrow \overline{f} \end{aligned} \end{equation*}
    all converge uniformly.
