Let \(\sum_{n=1}^{\infty}a_n\) be an absolutely convergent series. We’ll verify Cauchy’s criterion. Fix \(\varepsilon>0. \exists N\geq 1\) so that if \(n\geq m\geq N\) we have \(\sum_{k-m}^n|a_k|<\varepsilon\text{.}\) Now for all \(\geq m\geq N,|\sum_{k=m}^n a_k|\leq \sum_{k=m}^n|a_k|<\varepsilon\text{.}\) So \(\sum_{n=1}^{\infty}\) satisfies Cauchy’s criterion and hence converges.
