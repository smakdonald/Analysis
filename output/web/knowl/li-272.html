Item 1.

Given a set \(S\text{,}\) show that the function \(\rho_{\infty}: \mathcal{B}(S) \times \mathcal{B}(S) \rightarrow \mathbb{R}\) defined by
\begin{equation*} \rho_{\infty}(f, g):=\operatorname{lub}(\{|f(x)-g(x)|: x \in S\}) \end{equation*}
is a metric on \(\mathcal{B}(S)\text{.}\)
