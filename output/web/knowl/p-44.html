The field axioms imply the following statements, for any \(x, y,z \in F\text{.}\)
  1. \(\displaystyle -0=0\)
  2. \(\displaystyle -(-x)=x\)
  3. \(\displaystyle −(x + y) = −x − y\)
  4. \(xy = 0\) if and only if \(x = 0\) or \(y = 0\text{.}\)
  5. if \(xy = xz\) and \(x ≠ 0\text{,}\) then \(y = z\text{.}\)
  6. \(\displaystyle −(xy) = x(−y) = (−x)y\)
  7. \(\displaystyle (−1)x = −x\)
  8. \(\displaystyle (-x)(-y) = xy.\)
