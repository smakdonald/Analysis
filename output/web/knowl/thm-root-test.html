Theorem 3.10.

Consider \((a_n)_{n=1}^{\infty}\subseteq \mathbb{C}\)
  1. If \(\limsup_{n\rightarrow\infty}|a_n|^{\frac{1}{n}}<1\) then \(\sum_{n=1}^{\infty}a_n\) converges absolutely
  2. If \(\limsup_{n\rightarrow\infty}|a_n|^{\frac{1}{n}}>1\) then the series diverges.
  3. (if \(\limsup_{n\rightarrow\infty}|a_n|^{\frac{1}{n}}=1\) it could go either way)

Proof.
