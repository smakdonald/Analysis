Example 1.16.

  1. \(S=\mathbb{Q}\) set \(E=\{x\in\mathbb{Q}\,|\, x^2<2\}\text{.}\) Has neither sup nor inf (b/c working in \(\mathbb{Q}\text{,}\) not \(\mathbb{R}\)).
  2. \(S=\mathbb{Q}\) set \(E=\{x\in\mathbb{Q}\,|\, x>0\}\text{.}\) Then \(E\) has no sup, since not bounded above. \(\inf(E)=0\text{.}\)
