Definition 1.9. Order Relation.

A total order (or linear order) is a relation \(\leq\) on a set \(S\) that is:
  1. Reflexive: \(x\leq x\,\forall\, x\in S\)
  2. Transitive: \(x\leq y, y\leq z\implies x\leq z \, \forall\, x,y,z\in S\)
  3. Antisymetric: \(x\leq y, y\leq x\implies x=y\,\forall\, x,y\in S\)
  4. Total: \(x\leq y\) or \(y\leq x\,\forall\,x,y\in S\)
A totally ordered set is a set \(S\) in which an order is defined.
